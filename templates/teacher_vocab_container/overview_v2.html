FAQ

Q: What is this?

Even though this is not an official scientific theory, we believe that a student needs to learn a certain amount of words for each language level of the CEFR (Common European Framework of reference for language learning). The following list is a reference to how many words a student should know:

  • A0 -- 100
  • A1 -- 500
  • A2 -- 1000
  • B1 -- 2000
  • B2 -- 4000
  • C1 -- 8000

So what exactly is the Vocabulary Container? It is a software feature that lets the course teachers build the word lists for each CEFR language level.

Please note that in levels ranging from A0 to C1 you should add the words that you think are essential to the student. For example, adding 100 words of tree's biologic names are not essential. Specific jargon based vocabulary packs will be added later on based on the students choosing and they will be built on a different page, not this one. This Vocabulary Container is for essential words.
